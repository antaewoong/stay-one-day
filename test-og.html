3. Open Graph 검증 도구들

다음 도구들로 Open Graph가 제대로 설정되었는지 확인할 수 있습니다:

  • Facebook Open Graph Debugger
  • Twitter Card Validator
  • Open Graph Preview Tool
  • Hey Meta

테스트 방법:

  1. 위 도구들 중 하나를 열어주세요
  2. https://stayoneday.co.kr (또는 배포된 실제 도메인) 입력
  3. 미리보기가 제대로 나타나는지 확인

4. 카카오톡 테스트

카카오톡에서 실제로 테스트하려면:

  1. 카카오톡 대화방에 사이트 링크 입력
  2. 미리보기가 나타나는지 확인
  3. 만약 안 나타나면 캐시 문제일 수 있으니 몇 분 후 다시 시도

## 📊 전체 개발 진행율: **75%** 

---

## 🎯 프로젝트 개요

**Stay One Day**는 충청권(청주, 세종, 대전, 천안, 공주) 중심의 당일치기 숙소 대여 플랫폼입니다.

### 🔗 배포 정보
- **로컬 개발**: http://localhost:3000
- **GitHub**: https://github.com/antaewoong/stay-one-day
- **Vercel 배포**: 준비 중
- **데이터베이스**: Supabase PostgreSQL

## 🏗 시스템 아키텍처

### 📊 기술 스택
```
Frontend:  Next.js 14.0.4 + TypeScript + Tailwind CSS
Backend:   Supabase (PostgreSQL + Auth + Storage)
UI:        Lucide React Icons + shadcn/ui
Deploy:    Vercel (예정)
Version:   Git + GitHub
```

### 🗄 데이터베이스 스키마 현황
```sql
✅ accommodations (숙소 기본 정보)
✅ accommodation_images (숙소 이미지)
✅ accommodation_amenities (편의시설)
✅ reservations (예약 정보)
✅ users (사용자 정보)
⏳ reviews (리뷰 - 80% 완료)
⏳ wishlists (위시리스트 - 60% 완료)
```

### 🔍 **즉시 확인 가능한 기능들**
1. **메인 페이지** - http://localhost:3000
   - 히어로 슬라이드, 검색, 카테고리별 숙소
2. **숙소 목록** - http://localhost:3000/spaces
   - 필터링, 검색, 카드 디자인
3. **호스트 등록** - http://localhost:3000/host/register
   - 완전한 등록 폼
4. **숙소 등록** - http://localhost:3000/host/accommodations/new
   - 게시판 스타일 등록 시스템
5. **관리자 대시보드** - http://localhost:3000/admin
